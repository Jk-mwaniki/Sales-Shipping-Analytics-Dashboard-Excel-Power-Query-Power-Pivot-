**Sales And Shipping Anaysis Excel**

This project showcases an interactive Sales and Shipping Dashboard built using Microsoft Excel, leveraging Power Query and Power Pivot for data transformation and modeling.
It demonstrates my ability to transform raw data into actionable business insights using Microsoft Excel, Power Query, and Power Pivot.

I designed and built an interactive dashboard that enables stakeholders to quickly analyze sales performance, customer trends, and shipping efficiency—all in one place.

**Business Problem**

1.Organizations often struggle with:

2.Fragmented sales and shipping data.

3.Limited visibility into customer performance.

4.Inefficient reporting processes.

5.This dashboard solves these challenges by centralizing data and enabling real-time, self-service analysis.

**Key Impact**
1.Reduced manual reporting effort through automation

2.Enabled faster decision-making with interactive filters

3.Improved visibility into high-performing products and customers

4.Identified shipping inefficiencies across modes and priorities

**Technical Skills Demonstrated**

1.Data Preparation

2.Built ETL pipelines using Power Query

3.Cleaned and transformed raw datasets into analysis-ready formats

4.Data Modeling

5.Designed relational data models in Power Pivot

6.Created calculated measures using DAX

7.Data Visualization

8.Developed an interactive, user-friendly dashboard in Excel

9.Used charts, maps, and KPIs for clear storytelling

**Dashboard Highlights**
1.Sales Analysis
2.Monthly and quarterly sales trends (2014–2017)
3.Category-level performance (Bikes, Components, Accessories, Clothing)
4.Sales comparison across multiple representatives

**Customer Insights**

1.Top customers ranked by sales contribution

2.Individual performance tracking for key accounts

3.Shipping Performance

4.Average shipping cost per item by packaging type

5.Delivery time analysis by:
Shipping mode (Express Air, Regular Air, Delivery Truck)

6.Priority levels (Critical - Low)

**Geographic Insights**

State-level sales distribution
Regional shipping performance trends

**Interactivity**

The dashboard includes dynamic slicers for:
*Product Category
Salesperson
Year
Shipping Mode
Product Container*

This allows users to drill down into specific segments instantly.
